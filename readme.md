### Answer to the Questions
---
#### 1. What is the difference between var, let, and const?
- var is old style, has function scope, and can be redeclared and updated.
- let is newer, has block scope, can be updated but not redeclared in the same scope.
- const is like let but the value cannot be changed (constant). However, objects/arrays declared with const can have their contents changed.
#### 2. What is the spread operator (…)?
- The spread operator expands elements of an array or properties of an object. It’s useful for copying, merging, or passing elements individually.
#### 3. What is the difference between map(), filter(), and forEach()?
- map() transforms each element in an array and returns a new array with the results.
- filter() returns a new array with only the elements that pass a test condition.
- forEach() runs a function on each element but doesn’t return a new array.
#### 4. What is an arrow function?
- Arrow function is a shorter way to write functions using =>.
#### 5. What are template literals?#### 
- Strings wrapped in backticks that let embed variables or expressions easily using ${} and support multi-line strings.
